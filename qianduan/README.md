# 游戏陪玩后台管理系统 - 前端

基于 Vue.js 3 + Element Plus 开发的游戏陪玩后台管理系统前端项目。

## 项目介绍

这是一个为游戏陪玩后台管理系统设计的前端项目，包含以下功能模块：

- **陪玩人员管理**：陪玩人员信息录入、查询、修改、状态管理等
- **客户管理**：客户信息管理、消费统计、等级管理等
- **订单管理**：订单创建、状态管理、收入统计等
- **评价投诉管理**：评价管理、投诉处理等
- **财务管理**：提现管理、收入统计等
- **系统管理**：管理员、角色、数据字典等

## 技术栈

- Vue.js 3
- Element Plus UI组件库
- Vue Router 4
- Axios
- Vite

## 项目结构

```

├── utils/                  # 工具类
├── views/                  # 页面视图

│   ├── staff/              # 陪玩管理页面

│   ├── order/              # 订单管理页面
│   ├── evaluation/         # 评价管理页面
│   ├── complaint/          # 投诉管理页面
│   ├── finance/            # 财务管理页面
│   ├── system/             # 系统管理页面
│   ├── Home.vue           # 首页
│   ├── Login.vue          # 登录页
│   └── Layout.vue         # 主布局
├── App.vue
└── main.js
```

## 安装和运行

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## 页面功能说明

### 陪玩人员管理
- 陪玩人员信息的增删改查
- 技能等级、服务类型管理
- 状态管理（空闲、忙碌、离线、封禁）
- 收入和评分统计

### 订单管理
- 订单创建、查询、状态管理
- 订单流程控制（待支付→已支付→服务中→已完成）
- 收入统计和分析
- 客户和陪玩信息关联

### 客户管理
- 客户信息的增删改查
- 会员等级管理
- 消费统计和充值功能
- 状态管理

### 评价投诉管理
- 评价管理（星级评分、内容管理）
- 投诉处理流程
- 陪玩人员回复功能
- 统计分析

### 财务管理
- 财务统计报表
- 提现申请和审核
- 收入明细查询
- 余额管理

### 系统管理
- 角色权限管理
- 管理员账户管理
- 数据字典管理
- 系统配置

## API接口约定

所有API接口均以 `/api` 为前缀，返回统一格式的数据：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": 1234567890
}
```

## 开发规范

- 使用Vue 3 Composition API
- 遵循Element Plus组件规范
- 统一的错误处理机制
- 代码格式化使用ESLint和Prettier

## 注意事项

- 项目依赖后端API服务，请确保后端服务正常运行
- 所有数据均为模拟数据，实际使用时需连接真实API
- 生产环境部署时需配置API代理和环境变量
